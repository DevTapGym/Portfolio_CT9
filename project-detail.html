Project Overview

This is a mobile to-do list app designed to help users manage their personal tasks and collaborate in teams. It supports user registration and login (including Google login), email verification, task reminders, and daily progress tracking through a streak system and visual charts. The interface is clean, intuitive, and includes a dark mode option.

The project was built using Flutter for the frontend and Spring Boot for the backend. Key features include task categorization by priority, daily reminders, deadline-based search, group task assignment, and user customization in settings.

For more details on the source code and implementation, feel free to visit the project's GitHub repository.

Challenges & Solutions

Ensuring data consistency across screens

As the number of tasks increased, keeping data in sync across different screens became challenging and affected performance. I implemented state management (Bloc) to optimize data flow and ensure the UI remained consistently updated.

Designing a user-friendly interface with full functionality

The challenge was to create an intuitive interface while still supporting features like search, filter, and categorization. I applied a minimal UI design combined with clear logic structure to keep the user experience simple and efficient.
